# Restaurant Recommendation System

The Restaurant Recommendation System is a machine learning and NLP-based project designed to recommend restaurants based on user preferences. The system analyzes key restaurant attributes such as cuisines, price range, and ratings to provide personalized recommendations using a content-based filtering approach.

The project includes data preprocessing, exploratory data analysis (EDA), feature engineering, NLP-based text preprocessing, and restaurant embedding generation using Word2Vec. By leveraging cosine similarity, the system identifies restaurants that closely match a user's preferred cuisine, price category, and rating, resulting in more meaningful recommendations than traditional keyword-based methods.

In addition to the recommendation system, the project demonstrates essential machine learning workflows, including data cleaning, feature engineering, model development, and evaluation using Python and widely used data science libraries such as Pandas, NumPy, Scikit-learn, Gensim, Matplotlib, and Seaborn.

This project showcases practical applications of Natural Language Processing (NLP), recommendation systems, and machine learning while highlighting skills in data preprocessing, feature engineering, vector representations, and similarity-based recommendation techniques.
